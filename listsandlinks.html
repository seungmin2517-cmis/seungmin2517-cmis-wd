Instructions to Create a New Webpage

  1. You should clone your repository.
  2. Go to your repository file.
  3. First, you create a file called "(name).html" and "(name).css".
  4. Then, add the css link to html file. (link rel="stylesheet" href="blah.css" type="text/css")
  5. You edit your html file however you want to. Yet, it must start with "!doctype html". (Your html file may include, head, title, body, header, h, nav, main, p, footer, etc.)
  6. Once you are done with editing html file, go to css file so that you can edit it and make your html file colorful.
  7. When you edit your css file, you must start with { and end with }. For example: html{font-family: arial;}
  8. When you are done with editing css file, you should save both of them by going through git add, git commit -m "", and git push.
  9. Then you go to proudction server by the sftp command. Example: sftp seungmin2517@cmis-hsteacher. You must type your password, too.
